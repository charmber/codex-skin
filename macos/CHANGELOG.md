## 1.9.1 — 2026-07-18

### 修复

- 修复一次性 CDP 验证已经输出成功结果后，WebSocket 仍等待对端关闭而导致 Node 进程不退出、菜单栏长期卡在 `Skin ...` 的问题
- 修复后台脚本没有截止时间时，任一异常子进程都可能让原生菜单永久显示「正在准备主题引擎...」的问题

### 改进

- 一次性注入、验证与移除任务现在会在输出刷新后主动结束进程；长期运行的监视器收到停止信号后也会及时退出
- Shell 调用增加独立截止时间，原生菜单脚本增加进程组超时回收，避免残留的 Bash、Node 或系统弹窗子进程继续占用操作状态
- 热切换会强制清理拒绝退出的旧版本监视器，避免新旧两个注入器同时运行和反复覆盖主题
- watcher 改为优先由当前用户的 `launchctl` 托管，避免触发操作的菜单、Finder 或终端退出后监视器被一并回收
- 忙碌菜单会显示真实操作名称，非部署操作新增「取消当前操作」；发生超时后菜单会自动恢复并给出明确错误

### 说明

- 修复只调整本机主题引擎的进程生命周期，不改变回环 CDP 安全边界，也不修改官方 `.app`、`app.asar`、代码签名或 API 配置
